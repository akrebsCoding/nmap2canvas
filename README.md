A Tool to make a canvas for Obsidian from a nmap Scan (like "nmap -vv -sCV -A -oA scan 10.10.10.1")

Just copy/move the File in your /usr/local/bin Folder (mv nmap2canvas.py /usr/local/bin/nmap2canvas) 
Start the Tool in the Folder with your nmap xml file (nmap2canvas scan.xml)
